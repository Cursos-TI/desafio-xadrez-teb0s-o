# Movimentação de peças xadrez

---

## Funcionalidade:

Ao executar o programa, será exibida no terminal uma simulação dos movimentos das peças de xadrez, em que o **Bispo** realiza cinco deslocamentos consecutivos na diagonal superior direita; a **Torre**, cinco deslocamentos verticais para cima; a **Rainha**, oito deslocamentos horizontais para a esquerda; e o **Cavalo**, um deslocamento em "L", combinando um movimento vertical para cima seguido de um movimento horizontal para a direita.


## Compilação:

```bash
gcc -o xadrez xadrez.c
```
